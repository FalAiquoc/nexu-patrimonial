# Nexu Advogados - Blindagem Patrimonial

Landing page de alta conversao desenvolvida para o nicho juridico, com foco em protecao de ativos e sucessao familiar. O projeto foi estruturado para transmitir autoridade e seguranca, utilizando elementos visuais sobrios e uma copy estrategica baseada na exclusividade e eficiencia.

## Objetivo
O principal objetivo desta landing page e a captacao de leads qualificados (High Net Worth Individuals) interessados em solucoes juridicas de blindagem e planejamento sucessorio.

## Funcionalidades Principais
- Formulario de qualificacao dinamico.
- Design responsivo otimizado para dispositivos moveis.
- Secoes de servicos detalhadas (Holding Familiar, Testamento, Planejamento Tributario).
- Integracao facilitada com ferramentas de marketing e CRM.

## Integracoes Estrategicas
Este projeto foi desenhado para suportar integrações robustas:
- **CRM**: Envio de leads via Webhooks (n8n, Zapier) para sistemas como RD Station, HubSpot ou PipeDrive.
- **Meta (Facebook/Instagram)**: Implementacao de Meta Pixel para rastreamento de conversoes e criacao de publicos personalizados.
- **Canais de Atendimento**: Botoes diretos para WhatsApp e Telegram configurados para rastrear a origem do lead.
- **Analyticss**: Google Analytics 4 (GA4) para monitoramento de comportamento do usuario e funil de conversao.

## Compliance e Etica
O site segue rigorosamente as diretrizes do Codigo de Etica da OAB, mantendo o carater informativo e evitando a captacao indevida de clientela atraves de promessas de resultados ou mercantilizacao.
